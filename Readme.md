## Build, test and debug

1. Install all the dependencies
   `> yarn install`

2. Start local dev. server for the game and the UI
   `> yarn dev`

3. Run the tests for the game and the UI
   `> yarn test`

4. Build the game and the UI
    `> yarn build`

5. Start the game and the UI
    `> yarn start`
